# 단어와 표현 지침

어려운 말, 일본어 투 표현, 외국어, 번역 투, 띄어쓰기, 문장 부호, 주의 표현을 고칠 때 이 참고 파일을 사용한다.

## 1. 정확한 용어를 쓴다

- 개념을 정확히 표현하는 용어를 선택한다.
- 혼동이나 오해 가능성이 있는 단어는 피한다.
- 같은 문서 안에서는 같은 개념을 같은 용어로 쓴다.
- 유사어가 있으면 주체와 행위를 구분한다.
  - 예: 구직자는 `취업`하고, 기업은 `채용`한다.

### 2.1 줄임말과 약어를 남용하지 않는다

- 줄임말은 되도록 풀어 쓰고, 필요한 경우 처음 나올 때 원어 또는 풀이를 괄호에 넣는다.
- 외국 문자, 한자, 로마자 약어만으로 핵심 정보를 제시하지 않는다.

| 피할 표현 | 권장 표현 |
|---|---|
| AI 로봇을 이용한 지역 돌봄 | 인공지능(AI) 로봇을 이용한 지역 돌봄 |
| IoT 기반 대응 | 사물 인터넷(IoT) 정보 통신 기술(ICT) 기반 대응 |
| R&D 과제 발굴 | 연구·개발(R&D) 과제 발굴 |
| BM을 보유 | 사업 모델(BM)을 보유 |
| BRT 도로 구간 | 간선 급행 버스 / 간선 급행 버스 체계 도로 구간 |
| with 코로나 시대 | 단계적 일상 회복 시대 |

## 4. 번역 투를 피한다

| 피할 표현 | 권장 방향 |
|---|---|
| ~에 의해 ~되다 | 행위 주체를 주어로 쓴다. |
| ~을 통해 | `~로`, `~하여`, `~에서`, `~을 활용하여` 등 문맥에 맞게 바꾼다. |
| ~에 있다 | `~이다`로 바꾼다. |
| ~에 있어서 | `~에서`, `~에 관하여`, `~에 대하여`로 바꾼다. |
| 비교 시, 신청 시 | 비교할 때, 신청할 때 |
| ~하고 있는 | 단순 현재형 `~하는`으로 바꿀 수 있는지 검토한다. |
| ~되도록 하겠습니다 | `~하겠습니다`, `~하도록 하겠습니다` 등 책임 주체가 드러나게 쓴다. |
| ~에 대한 | `~의`, `~에 관한`, `~을 위한` 등 문맥에 맞게 바꾼다. |

나쁜 예:

> 조선은 태조 이성계에 의해 건국되었다.

좋은 예:

> 태조 이성계가 조선을 건국했다.

나쁜 예:

> 우리의 목표는 서비스 안정화에 있다.

좋은 예:

> 우리의 목표는 서비스 안정화이다.

## 6. 문장 부호와 인용

- 법률명, 조례명, 규정명은 `「 」`를 사용한다.
- 강조가 필요한 명칭, 행사명, 구호 등은 작은따옴표(`‘ ’`)를 우선 사용한다.
- 직접 인용문은 마침표를 인용 부호 안에 넣고, 필요한 경우 `라고`를 붙인다.
  - 예: `“추진하겠습니다.”라고 밝혔다.`
- 공문서의 마지막에는 필요한 경우 `끝.`을 쓴다.

## 7. 금지 또는 주의 표현

아래 표현은 문맥상 꼭 필요한 경우가 아니면 바꾼다.

| 주의 표현 | 권장 표현 |
|---|---|
| 만전을 기하여 주시기 바랍니다 | 철저히 준비해 주시기 바랍니다, 소홀함이 없도록 해 주시기 바랍니다 |
| 착오 없으시기 바랍니다 | 확인해 주시기 바랍니다 |
| 적극 협조 바랍니다 | 협조해 주시기 바랍니다 |
| 통보합니다 | 알려 드립니다 |
| 요망합니다 | 바랍니다, 요청드립니다 |
| 문의 바람 | 문의하시기 바랍니다 |
| 표시하기 바람 | 표시해 주시기 바랍니다 |
| 복용할 것 | 복용하세요, 복용해야 합니다 |
| 갖출 것 | 갖추어야 합니다 |
| 불이익을 받을 수 있습니다 | ○○하지 않으면 ○○가 제한됩니다 |
| 처리 방안 부재 | 처리 방안이 없음, 처리가 어려움 |
| 비교 시 | 비교할 때 |
| 신청 시 | 신청할 때 |
| 운영하고 있는 | 운영하는 |
| 추진 중에 있으며 | 추진하고 있으며 |
| 실시한다 | 한다, 시행한다 |

### 7.1 국민 관점에서 완곡하게 표현한다

- 명령형 `-ㄹ 것`, `-기 바람`, `-기 바랍니다`의 남용은 고압적으로 보일 수 있으므로 문맥에 맞게 `-하세요`, `-해야 합니다`, `-해 주시기 바랍니다`로 바꾼다.
- 권위적 표현을 고칠 때 법적 의무, 금지, 제재의 강도는 임의로 약화하지 않는다.
- 특정 집단을 차별하거나 낙인찍는 표현, 누구나 알기 어려운 정책 용어, 문맥상 필요 없는 인적 정보는 줄인다.
